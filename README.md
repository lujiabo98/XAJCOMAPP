# XAJCOMAPP
调用XAJ仓库中写成的新安江模型COM组件

在用C++以面向对象（OOP）形式编写三水源新安江模型的基础上，进一步将其写成组件形式。

这里是调用`XAJ.dll`的一个示例程序，并不包含`XAJ.dll`。`XAJ.dll`位于XAJ仓库。
